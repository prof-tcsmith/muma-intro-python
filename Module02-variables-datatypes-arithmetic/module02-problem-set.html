Problem 5: Inventory Planning with Modulo and Floor Division

Task:
A warehouse manager needs to plan shipping containers:

  1. Create a variable total_items with a value of 237

  2. Create a variable items_per_container with a value of 40

  3. Use floor division (//) to calculate how many complete containers can be filled

  4. Use modulo (%) to calculate how many items will be left over

  5. Print the results in a message that would help the warehouse manager plan

Hint Floor division (//) gives you the number of times one number fits completely into another, discarding any remainder. Modulo (%) gives you the remainder. For example, 237 // 40 = 5 and 237 % 40 = 37.

Problem 6: Order of Operations in Profit Calculation

Task:
Calculate the profit for a product line:

  1. Set revenue to 120000 and cost_of_goods to 65000

  2. Set overhead_expense to 15000 and marketing_expense to 10000

  3. Calculate profit by subtracting all expenses from revenue

  4. Create an expression that adds a 5% bonus to the profit, but only on the portion of profit that exceeds 25000

  5. Print the results of both calculations

Hint

First calculate the basic profit. Then for the bonus, you need to:

  1. Calculate the profit exceeding 25000: max(profit - 25000, 0)

  2. Calculate 5% of this excess: 0.05 * max(profit - 25000, 0)

  3. Add this bonus to the profit: profit + (0.05 * max(profit - 25000, 0))

Use parentheses to ensure the operations happen in the correct order.

Problem 11: Loan Ceiling Calculation

Task:
A bank needs to calculate the maximum loan amount:

  1. Import the math library

  2. Set monthly_income to 5000

  3. Set debt_to_income_ratio to 0.3 (30% of income can go to loan payment)

  4. Set interest_rate to 0.05 (5% annual interest)

  5. Set loan_term_years to 30

  6. Convert the annual interest rate to monthly: monthly_rate = interest_rate / 12

  7. Calculate the total number of payments: num_payments = loan_term_years * 12

  8. Calculate the maximum monthly payment: max_payment = monthly_income * debt_to_income_ratio

  9. Use the loan formula to calculate the maximum loan amount: max_loan = max_payment * ((1 - (1 + monthly_rate) ** -num_payments) / monthly_rate)

  10. Use math.pow() for the exponentiation

  11. Use math.floor() to round down to the nearest dollar (conservative approach)

  12. Print the maximum loan amount with clear labeling

Hint This is a more complex problem using a standard mortgage formula. The key is to use math.pow() for the exponentiation part: math.pow((1 + monthly_rate), -num_payments). The formula calculates the present value of an annuity, which is the maximum loan amount based on what monthly payment the borrower can afford.

Problem 12: Inflation-Adjusted Investment Calculation

Task:

Solution 8: Break-Even Analysis

import math

# Set variables
fixed_costs = 50000
price_per_unit = 99.99
variable_cost_per_unit = 40.50

# Calculate contribution margin
contribution_margin_per_unit = price_per_unit - variable_cost_per_unit

# Calculate break-even point
break_even_units = fixed_costs / contribution_margin_per_unit
# Round up since we can't sell partial units
break_even_units_rounded = math.ceil(break_even_units)

# Calculate break-even revenue

# Print results
print("===== BREAK-EVEN ANALYSIS =====")
print("Fixed Costs: $", fixed_costs)
print("Price per Unit: $", price_per_unit)
print("Variable Cost per Unit: $", variable_cost_per_unit)
print("Contribution Margin per Unit: $", round(contribution_margin_per_unit, 2))
print("-------------------------------")
print("Break-Even Point:", break_even_units_rounded, "units")
print("Break-Even Revenue: $", round(break_even_revenue, 2))
print("===============================")

Solution 11: Loan Ceiling Calculation

import math

# Set variables
monthly_income = 5000
debt_to_income_ratio = 0.3  # 30%
interest_rate = 0.05  # 5%
loan_term_years = 30

# Convert annual interest rate to monthly
monthly_rate = interest_rate / 12

# Calculate total number of payments
num_payments = loan_term_years * 12

# Calculate maximum monthly payment
max_payment = monthly_income * debt_to_income_ratio

# Calculate maximum loan amount using loan formula
# Formula: PV = PMT * ((1 - (1 + r)^-n) / r) where:
# PV = present value (loan amount)
# PMT = payment
# r = monthly interest rate
# n = number of payments
denominator = monthly_rate
numerator = 1 - math.pow((1 + monthly_rate), -num_payments)
max_loan = max_payment * (numerator / denominator)

# Round down to nearest dollar
max_loan_rounded = math.floor(max_loan)

# Print results
print("Loan Ceiling Calculator:")
print("Monthly Income: $", monthly_income)
print("Maximum Payment (30% of income): $", round(max_payment, 2))
print("Loan Term:", loan_term_years, "years")
print("Annual Interest Rate:", interest_rate * 100, "%")
print("------------------------------")
print("Maximum Loan Amount: $", max_loan_rounded)

Solution 12: Inflation-Adjusted Investment Calculation

import math

# Set variables
principal = 10000
investment_return_rate = 0.08  # 8%
inflation_rate = 0.03  # 3%
years = 20

# Calculate nominal future value
nominal_value = principal * math.pow((1 + investment_return_rate), years)

# Calculate inflation factor
inflation_factor = math.pow((1 + inflation_rate), years)

# Calculate inflation-adjusted value
real_value = nominal_value / inflation_factor

# Round to 2 decimal places
nominal_value_rounded = round(nominal_value, 2)
real_value_rounded = round(real_value, 2)

# Print results
print("Investment Future Value Calculator:")
print("Initial Investment: $", principal)
print("Annual Return Rate:", investment_return_rate * 100, "%")
print("Annual Inflation Rate:", inflation_rate * 100, "%")
print("Investment Period:", years, "years")
print("------------------------------")
print("Nominal Future Value: $", nominal_value_rounded)
print("Inflation-Adjusted Value: $", real_value_rounded)
print("Purchasing Power Loss: $", round(nominal_value - real_value, 2))
